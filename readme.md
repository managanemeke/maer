## maer

### What were the versions used?

- Ubuntu 20.04
- PHP 7.4.3
- Composer 1.10.1
- PhpStorm 2023.3.2

### How to run?

- Clone repository.

```
git clone https://github.com/managanemeke/maer.git && cd maer
```

- Install dependencies and generate autoload.php.

```
composer install
```

- Run script.php.

```
php script.php input.csv output.csv
```

- Check the output contents.

```
cat output.csv
```

```
3,4,6,9,1
```
